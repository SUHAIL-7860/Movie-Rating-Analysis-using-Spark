# 🎬 Movie Rating Analysis using Spark

This project analyzes movie rating data using PySpark to identify the most popular movies based on user engagement.

---

## 🚀 Features
- Analyze large-scale movie data using PySpark
- Identify most popular movies based on rating count
- Perform groupBy and aggregation operations
- Understand real-world big data processing

---

## 🛠 Tech Stack
- Python
- PySpark
- Google Colab

---

## 📊 Key Insight
More ratings = More popularity

---

## 📂 Project Files
- movie_rating_analysis.ipynb

---

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Install PySpark if required
3. Run all cells

---

## 💡 Real-World Use
This type of analysis is used by platforms like Netflix, Amazon, and YouTube to understand user preferences.
